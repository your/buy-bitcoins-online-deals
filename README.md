# Top 100 cheapest buy-bitcoins-online deals finder

## Setup

```
bundle install
```

Note: You can add a notification email, look at the source code.

## Example usage

Add to cron (e.g. every day at 19.00):

```
echo "0 19 * * * path/to/this/script.rb >/dev/null 2>&1" | crontab -
```
